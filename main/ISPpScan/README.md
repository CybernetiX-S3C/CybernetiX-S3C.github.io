# ISPpScan or ISP Protect Scan
## Author: John Modica @ CybernetiX S3C

ISP Protect Scan is a tool that scans the ISP info and checks for vulnerabilities.

```markdown
# Download:

git clone https://github.com/cybernetix-s3c/ISPpScan

# How To Use:

cd ISPpScan
./ispp_scan.sh

```
