##IP Locator
###Author : John Modica @ CybernetiX S3C

No need for installation. Just run it with perl.
```markdown
Usage  : ./Iplocation.pl [host] [ip] [domain] 

Ex:  ./Iplocation.pl  www.google.com 
     ./Iplocation.pl  216.58.210.206
 ```
